Windows Community Toolkit

Overview

Windows Community Toolkit is a Fluent UWP app that I’ve helped drive to success from it’s beginning. This app aims to hit many business and user requirements some being community engagement, growing UWP dev base and helping new Windows platform developers learn XAML.

Adding value

I wanted the app to not only provide useful code snippets but also be a place where designer and developers can learn about XAML and the Microsoft design language.

Properties

Designers and learning developers can interact with each control without touching the code and learn its behavior. They can also jump into the code to get a better understanding of how their designs are implemented from a developers standpoint, even I found myself digging inside the code when I wanted to understand how shadows are implemented in XAML.

Usability studies

My team and I kept a close relationship to our user base which constantly provided us with fantastic feedback that we then integrated into our usability studies. We preformed usability studies on existing and new features that yielded incredible insight into what was working and what needed to be improved. All this effort resulted in positive feedback from our community which can be seen with a overwhelming positive review on the app store and Github.

Redesining samples

Addition to our ongoing design process on the app itself, I redesigned or updated many of the controls/samples used in the app.
